##############################################################################
#                           PROGRAM DESCRIPTION                              #
##############################################################################


The program "6936HW4" reads strings from a text file, inserts them into linked
 list nodes, generates a sublist, and sorts the sublist according to
 specification provided in CSCI 2421-E01 HW4 Spring 2016.



##############################################################################
#                               SOURCE FILES                                 #
##############################################################################


Name:  main.cpp
   Main program.  This is a driver for the three functions needed to
   accomplish the tasks given in the instructions.  More details can be found
   in the source code.

Name:  node1.h
   This is the declaration header for the node class.

Name:  node1.cpp
   This is the implementation file for the node class.

Name:  list_demo.h
   This is a declaration header for functions in the list_demo namespace.

Name:  list_demo.cpp
   This is an implementation file for functions in the list_demo namespace.

Name:  data.txt
   This is a sample data.txt data file.  You may utilize a custom data file of
   any name if you wish.
   
Name:  Makefile
   This facilitates trouble-free compilation of the program.
   
Name:  README.txt
   This is the file you are currently reading.
   You must have done something correctly.
